## K8s App
This is a very simple app for demonstrating how k8s works together with other tools. This is intended for beginners 

<h4>How to Use</h4>
- Run the file <strong>create_repo.sh</strong> in the terminal
- Run like so: 

    ```sh
        sudo chmod u+x create_repo.sh
        sudo ./create_repo.sh 
    ```
- Push to the repo
